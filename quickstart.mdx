## 1. Create an API key

Go to **Settings → Developer → API Keys** in your Aasure dashboard and click **New API key**.

Give it a name (e.g. `my-integration`), select the scopes you need, and click **Create**. Copy the token immediately — it is shown only once.

Your token looks like:

```
aas_live_AbCdEfGhIjKlMnOpQrStUvWx
```

<Warning>
  Store the token securely. It cannot be retrieved after creation — only revoked and replaced.
</Warning>

## 2. Make your first request

Replace `YOUR_API_KEY` and `YOUR_BUSINESS_ID` below:

<CodeGroup>

```bash cURL
curl https://biz.aasure.app/api/v1/businesses/YOUR_BUSINESS_ID/api-keys \
  -H "Authorization: Bearer YOUR_API_KEY"
```

```typescript TypeScript
const response = await fetch(
  'https://biz.aasure.app/api/v1/businesses/YOUR_BUSINESS_ID/api-keys',
  { headers: { Authorization: 'Bearer YOUR_API_KEY' } }
);
const data = await response.json();
console.log(data.api_keys);
```

```python Python
import httpx

client = httpx.Client(headers={"Authorization": "Bearer YOUR_API_KEY"})
r = client.get(
    "https://biz.aasure.app/api/v1/businesses/YOUR_BUSINESS_ID/api-keys"
)
print(r.json()["api_keys"])
```

</CodeGroup>

You should get a `200` response with your active API keys.

## 3. Handle errors

Every error has the same shape. Check `error.code` to branch:

```typescript
const res = await fetch(url, { headers });
if (!res.ok) {
  const { error } = await res.json();
  console.error(`[${error.code}] ${error.message}`);
  // error.request_id is useful when contacting support
}
```
